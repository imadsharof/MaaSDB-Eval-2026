# INFO-H509 : Mobility Data Analysis using LLMs 🚢🤖

## 📌 Description du projet
Ce projet a été réalisé dans le cadre du cours de **Geospatial Web Technologies (INFO-H509)** à l'École Polytechnique de Bruxelles. 
Notre objectif est d'évaluer les capacités des Grands Modèles de Langage (LLMs) récents (comme Claude 3 Opus et Gemini 1.5/2.5) à analyser des données de mobilité spatio-temporelles complexes (données AIS de navires).
